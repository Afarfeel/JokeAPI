Usage:

To use the JokeAPI, send a GET request to https://sv443.net/jokeapi and make sure you provide the joke_category header. Otherwise this HTML page will appear.
The content of the joke_category header has to correspond to any of the following joke categories (case sensitive!):
  • Any        (returns a joke from any of the following categories)
  • Programming
  • Miscellaneous
  • Dark        (dark humor)

! >    If you know any jokes that you want me to add, please submit them using this form. Thanks   : )








Formatting:

The formatting is very simple: {
    "category": "The category of the joke",
    "type": "single or twopart",
    "joke": "this property only exists if the type property is single",
    "setup": "this property is the setup of a joke with the type twopart",
    "delivery": "this property is the delivery of a joke with the type twopart",
    "nsfw": "will be true if a joke is potentially not safe for work or will not exist if not"
    "religious": "will be true if a joke is offensive against a religion or will not exist if not"
    "political": "will be true if a joke is potentially political or will not exist if not"
}























Example with CURL:

curl -X GET "https://sv443.net/jokeapi" \
     -H "joke_category:Programming" This will return a programming joke





Example with NodeJS:

var randomJoke;

http.get({
    host: "https://sv443.net",
    path: "jokeapi",
    headers: {
        "joke_category": "any"
    }
}).then(res => {
    randomJoke = JSON.parse(res);
});
 The variable randomJoke is now a JSON object like the one at the very top of this page





Example with jQuery AJAX:

var randomJoke;

$.ajax({
    method: "GET",
    url: "https://sv443.net/jokeapi",
    headers: {
        "joke_category": "any"
    },
    success: (data, status) => {
        randomJoke = JSON.parse(data);
    }
});
 The variable randomJoke is now a JSON object like the one at the very top of this page
